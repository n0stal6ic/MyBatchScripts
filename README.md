# MyBatchScripts
Repository full of useful batch scripts I have created over time for various purposes.


- Music2YT (Script that utilizes ffmpeg to convert Image & Music to a raw, high quality format for good looking youtube music video uploads)
- MediaDL (Script that utilizes ffmpeg and YT-DLP to download & transcode various formats from youtube, including best.)
- SoundCloudDL (Script that utilizes YT-DLP to download and attach metadata for soundcloud tracks in Go+ 256kbps AAC (wrapped in M4A container) provided valid cookies.)
- ArcRes (Unfinished IPTV script that attempts to restream a HLS stream to a reboadcasted MP4 indefinitely until stopped.)
- OVRMove (Moves the OpenVR folder to the desktop, and back to appdata, so roblox doesn't launch in VR mode.)
- WinTV2VRC (Unfinished script that attempts to take a WinTV HVR-950 Dongle and restream it to a HLS stream perfect for VRChat)
- MKVMux (Lets you take individual video, audio, and substitle files to mux into a single MKV with ffmpeg.
